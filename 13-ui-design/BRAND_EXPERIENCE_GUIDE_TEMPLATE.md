# Brand Experience Guide Template

**목적**: 각 프로젝트가 자신의 Brand Experience Guide를 작성할 때 사용하는 템플릿.
         이 파일 자체에 값을 채우지 말 것 — 프로젝트별 docs 폴더에 복사 후 작성.

**위치**: `docs/<project>-constitution/BRAND_EXPERIENCE_GUIDE.md`
**관계**: 14-experience-direction/EXPERIENCE_CONSTITUTION.md §1–§7 7개 축의 실제 값

---

## 작성 필수 항목 (7개 축)

### 1. Brand Emotion Axis
감정 차원 3–7개를 0.0–1.0 강도로 정의한다.
포지셔닝 문장: "이 브랜드는 [A]하지만 [B]하지 않고, [C]하면서도 [D]하다"

### 2. Visual Tone
색상 온도, 채도, 대비 수준을 정의한다.
"따뜻함/차가움", "절제됨/풍부함" 축으로 위치를 표시한다.

### 3. Typography Personality
서체가 전달하는 성격을 정의한다. 단순 font 이름이 아니라 감성 어휘로 기술한다.

### 4. Image Treatment Philosophy
사진/일러스트의 처리 방식: 색보정 기준, 합성 스타일, 감정 온도, 금지 스타일을 정의한다.

### 5. Space and Rhythm
여백의 철학: "공간이 메시지다" vs "밀도가 신뢰다" 같은 포지션을 정의한다.

### 6. Motion Philosophy
움직임이 브랜드에 어떤 역할을 하는가: 주목, 전환, 피드백, 기쁨 중 무엇을 우선하는가.

### 7. Microcopy Tone
버튼 레이블, 에러 메시지, 빈 상태 문구의 톤: formal/casual, active/passive, warm/neutral.
